# Python en Español — El Profesor

## Deploy en Vercel

1. Sube estos archivos a GitHub
2. Conecta en vercel.com
3. En Settings → Environment Variables agrega:
   - REACT_APP_ANTHROPIC_API_KEY = tu key de Anthropic
