TypeError en Python

¿Estás mezclando tipos de datos y Python te regaña? Te explicamos en español qué pasó y cómo arreglarlo.

¿Qué es un TypeError?

Ocurre cuando intentas hacer una operación con tipos de datos incompatibles. Como sumar un número con un texto — Python no sabe qué hacer con eso.

❌ Error:

nombre = "Diego" edad = 25 print(nombre + edad) # TypeError: can only concatenate str to str

✅ Correcto:

nombre = "Diego" edad = 25 print(nombre + " tiene " + str(edad) + " años") # o mejor aún: print(f"{nombre} tiene {edad} años")

¿Por qué pasa?

Las causas más comunes son mezclar texto con números, llamar algo que no es función, o usar un método en el tipo de dato equivocado.
